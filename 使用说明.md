# 个人网站使用说明

## 文件说明

- `index.html`：前台展示页
- `admin.html`：后台维护页
- `styles.css`：样式文件
- `storage.js`：默认内容与本地存储逻辑
- `app.js`：前台渲染逻辑
- `admin.js`：后台维护逻辑

## 你现在怎么用

1. 打开 `admin.html`
2. 在网页里直接修改姓名、研究方向、项目、成果、联系方式
3. 点击“保存修改”
4. 再打开 `index.html` 查看效果

## 导入导出

- 如果想备份网站内容：点“导出 JSON”
- 如果想恢复内容：在后台页点“导入 JSON”

## 当前版本的维护方式

当前版本默认把数据保存在浏览器本地，所以：

- 同一台电脑、同一浏览器里维护最方便
- 如果换电脑或换浏览器，建议先导出 JSON 备份
